⚙️ Configurar Supabase
Siga os passos abaixo para conectar o banco de dados. Feito uma vez, vale para sempre.
1
Criar conta e projeto no Supabase
Acesse supabase.com, crie uma conta gratuita e clique em "New Project". Escolha um nome (ex: arbus-academy) e uma senha forte para o banco.
2
Criar a tabela de progresso
No painel do Supabase, vá em SQL Editor e execute o SQL abaixo:
-- Tabela de progresso por usuário CREATE TABLE progress ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, user_id uuid REFERENCES auth.users(id) ON DELETE CASCADE, module_id integer NOT NULL, percent integer DEFAULT 0, completed boolean DEFAULT false, updated_at timestamptz DEFAULT now(), UNIQUE(user_id, module_id) ); -- Permitir que cada usuário veja só seu progresso ALTER TABLE progress ENABLE ROW LEVEL SECURITY; CREATE POLICY "users_own_progress" ON progress FOR ALL USING (auth.uid() = user_id);
3
Pegar as credenciais do projeto
No painel do Supabase, vá em Project Settings → API. Copie a Project URL e a chave anon public e cole abaixo.
