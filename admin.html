⚙️ Configurar Supabase
Conecte o painel ao banco de dados do Arbus Academy.
1
Acesse supabase.com, abra seu projeto arbus-academy e vá em SQL Editor. Execute o SQL abaixo para adicionar a view de admin:
-- View para o painel admin ver todos os usuários e progresso CREATE OR REPLACE VIEW admin_user_progress AS SELECT u.id as user_id, u.email, u.raw_user_meta_data->>'full_name' as full_name, u.created_at, COALESCE(AVG(p.percent), 0)::int as avg_progress, COUNT(p.id) as modules_started, COUNT(CASE WHEN p.completed THEN 1 END) as modules_done, MAX(p.updated_at) as last_activity FROM auth.users u LEFT JOIN public.progress p ON p.user_id = u.id GROUP BY u.id, u.email, u.raw_user_meta_data, u.created_at; -- Permitir que admins vejam a view GRANT SELECT ON admin_user_progress TO authenticated; -- View detalhada por módulo por usuário CREATE OR REPLACE VIEW admin_module_progress AS SELECT p.*, u.email, u.raw_user_meta_data->>'full_name' as full_name FROM public.progress p JOIN auth.users u ON u.id = p.user_id; GRANT SELECT ON admin_module_progress TO authenticated;
2
